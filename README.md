# CH585M 多模态交互智能手环

本仓库为嵌入式设计大赛项目代码提交仓库。

## 项目简介

项目基于沁恒 CH585M RISC-V 蓝牙主控，实现多模态交互智能手环：

- BLE HID 空中飞鼠
- BMI270 六轴姿态控制鼠标移动
- PA3 / PA6 实体按键左键、右键与 PPT 翻页
- MAX30100 心率检测
- 高心率屏幕报警与 PB6 振动提醒
- ST7789 屏幕状态显示
- 触摸屏页面切换与模式切换
- 共享 I2C 总线恢复与触摸优先调度

## 工程目录

- `code/`：CH585M 固件工程源码与工程配置
- `code/obj/code.hex`：当前 V1.0 可烧录固件

## 当前版本

V1.0：已完成 BLE HID 空中飞鼠、心率显示、高心率报警、屏幕交互、PPT 翻页与 I2C 稳定性优化。
